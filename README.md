# 🚗 Sistema de Controle de Vistorias

Este projeto foi desenvolvido em **Python** para automatizar o fluxo de vistorias veiculares, aplicando regras de negócio para identificação de irregularidades e geração de estatísticas operacionais.

## 🛠️ Funcionalidades
- **Entrada de Dados:** Processamento de múltiplos veículos em um único fluxo.
- **Regra de Negócio (Ano):** Alerta automático para veículos fabricados antes de 2010 (verificação de chassi).
- **Validação Técnica:** Critérios de aprovação baseados no estado dos pneus.
- **Relatório Geral:** Cálculo automático de média de idade da frota e porcentagem de aprovação/reprovação.

## 💻 Tecnologias e Conceitos
- **Linguagem:** Python 3.14
- **Conceitos Aplicados:** Estruturas de repetição (`for`), condicionais aninhadas, acumuladores e tratamento de divisão por zero.
- **Nível:** Projeto consolidado do Mundo 2 (Curso em Vídeo) com foco em aplicação real.
